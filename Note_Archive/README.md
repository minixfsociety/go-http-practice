# 📝 Go Note Archive API

A simple REST API for managing notes in memory. Built to practice Go's `net/http` package and JSON handling.

## 🚀 How to Run

1. Open terminal and run:
   ```bash
   go run main.go

2. The server will start on http://localhost:8080.

3. Method	Endpoint	Description
   GET	    /notes	    Get a list of all notes
   POST	    /add-note	Add a new note (JSON body required)
